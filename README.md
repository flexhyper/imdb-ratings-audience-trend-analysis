# IMDb Ratings and Audience Trend Analysis

## Overview
This project analyzes large-scale IMDb movie rating data to study audience sentiment, genre-based engagement, and long-term rating patterns in the entertainment industry.

The study uses rating-based sentiment classification as an alternative to text-based sentiment analysis, especially in scenarios where textual review data is limited or unavailable.

## Research Context
This work was developed as part of an academic analytics study and later published as an **IEEE publication**.

## Objectives
- Analyze audience sentiment using structured movie rating data
- Identify rating trends across time and genres
- Study engagement behavior across different categories of films
- Explore the use of large-scale rating data for entertainment analytics

## Dataset Scope
- Nearly **1 million IMDb ratings**
- Coverage across multiple movie genres
- Focus on rating distributions, audience sentiment classes, and engagement patterns

## Key Findings
- Ratings showed a **declining trend over time**
- Significant **genre-wise differences** were observed in audience sentiment
- Engagement patterns varied across movie categories
- Structured big data proved useful for supporting content strategy and audience analysis
